# web-chat
 A real time web chat using golang

## Rodando a aplicação

Execute o comando <code>docker-compose up build</code>

## Consumindo a api

### Rotas (GET)
- localhost:3333/users
- localhost:3333/users/{ID}

### Rotas (POST)
- localhost:3333/users

### Rotas (PATCH)
- localhost:3333/users/update

### Rotas (DELET)
- localhost:3333/users/remove/{ID}
